# Chat

## Requisitos
- Boost C++ Libraries </br>
- Servidor MySQL 
- MySQL Connector/C++

## Instalação
### Boost C++ Libraries
```#apt-get update && apt-get install libboost-all-dev```

### MySQL Server
```#apt-get update && apt-get install mysql-server```

### MySQL Connector/C++
```#apt-get update && apt-get install libmysqlcppconn-dev```
